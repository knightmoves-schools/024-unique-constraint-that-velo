The SQL in the `exercise.sql` file
- Should create a table named `Products` with the following columns; `ID`, `PRODUCT_NAME`, `SERIAL_NUMBER`, `CATEGORY`, and `PRICE`
- Should make the `SERIAL_NUMBER`column unable to repeat values
- Should make the following columns require values; `PRODUCT_NAME`and `PRICE`
